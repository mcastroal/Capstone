# Muay Thai Training Tracker

## Project Description
A web application that allows Muay Thai athletes to track training sessions and receive AI-generated insights based on their training history.

## Tech Stack
React (Next)
Node.js / Express
MySQL
OpenAI API
AWS (S3, EC2, RDS)

## Pages
1. Dashboard
2. Training Log
3. AI Insights

## User Roles
User
Admin(coach)

## AI Feature
Analyzes training sessions and suggests improvements.
